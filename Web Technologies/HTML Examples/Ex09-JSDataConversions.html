Points to remember:

  • All inputs from the HTML elements will be in the form of strings
  • There are 2 APIs to convert strings to numbers
  • parseInt fn is used to convert a string to a integer number
  • parseFloat fn is used to convert a string to a floating number
  • U can use isNan to check for the number input. If the string is not a valid number, then it will be called as NaN(Not a Number). It is recommended to use Number.isNaN instead of the global isNaN method.
