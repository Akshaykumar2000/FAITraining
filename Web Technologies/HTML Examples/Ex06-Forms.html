Points to remember:

  • Forms are the way of taking user inputs in html.
  • Forms allow to create input tags to take the inputs that could be text to email to phoneno or any other kind including fileName
  • It has 2 attributes called action and method. action is used to mention the file that processes the form contents when submited. The method will take values like POST, PUT, GET to define the direction of the data transfer from the form.
  • Form will have a submit button that is clicked after the entries are filled by the User. This processes the form based on the action value set.
  • For client side programming, action attribute does not make any sense as it is used only for server side programming.
  • Input type can have any of the following values:
    1. Text
    2. Number
    3. Email
    4. Date/DateTime
    5. File
    6. Search
    7. Button
    8. CheckBoxes
    9. Radio
    10. range
