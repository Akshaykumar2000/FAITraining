Points to remember

  1. Function declaration in JS is done by using function keyword.
  2. All data type declarations are let here, so no need to give the declaration keywords like var, let or const
  3. Return type of the function is not specified. The function will return a value only if it has return statement in its implementation
  4. Functions can also return an object or an array in JS
  5. Functions can be anonymous/lambda based.
