Image files in HTML

Phaniraj B.N.




Points to remember

  • Images can be placed in the HTML Pages using img tag.
  • Its src attribute is used to set the path of the file that U want to associate with the img.
  • If the Image is not able to load, U can use alt attribute to set an alternate text inplace of the img.
  • It is recommended to use the relative path for the files as it is easy to maintain
