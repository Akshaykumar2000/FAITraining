Data types in JavaScript

Points to remember:

  • There are 3 keywords to declare data in JS: var, let and const.
  • var is the older version which is no longer recommended. let and const are the prefered ones. const is used to store the data that does not change. let is used to store variables. They are all local variables.
  • JS is scope based. The let is more scope based compared to var. var is now obselete. However, for backward compatibility var is still supported by the newer JS interpretors.
  • variables declared but unassigned are called as undefined data. Any variable that is refered without assignment are called undefined data
  • PS: The output of the script is displayed in the console of the browser.
  • U can use interpolation syntax for creating string literals with variable data. We use backlit key to start the string and end it with backlit again
