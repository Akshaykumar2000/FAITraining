JS Basics

Points to remember:

  • JS is a client side scripting language used to create dynamic web content
  • It is the default scripting language for all browsers.
  • As the JS code is executed in the browser, it is called as Client side scripting
  • JS is open source and is maintained by ECMA(European Computers Manufacturers Association). Latest version is ECMA Script 11
  • JS is a simple yet powerfull scripting language that allows U to create dynamic content, store data in memory and perform complex operations for a Dynamic web content
  • JS is supported by various Frameworks for creating dynamic and powerfull web content. This includes Angular from GOOGLE, React from FACEBook, Nodejs from Joyant Tech.
  • A scripting language is one that is interpreted by a thin software like a Browser. It is different from Compiler languages in terms of weight and syntax. JS is one of the popular scripting languages in the computing world.
  • JS has features like Object based Programming, Collection Classes, UI based DOM Manipulations, Functions, CallBack Functions and many more
  • With new ECMA script, JS comes with lot of features that enhance the UI Experience for UR Web Apps.
  • Scripting code is included in the HTML document and will be executed. We use script tag to insert the JS code within the HTML Document.
  • U can place the script tag anywhere in the document, but it is recommended to place it just above the closure of the body tag.
  • JS comes with 3 UI based functions to perform UI interactions:alert(), prompt() and confirm()
