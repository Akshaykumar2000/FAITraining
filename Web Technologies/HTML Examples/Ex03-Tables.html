Points to remember

  • table is the tag used to create tables in html
  • U can divide the table into thead, tbody and tfoot
  • A row is represented using a tag called tr
  • A cell in the row is represented using a tag called td or th
  • U can use styles like width, text-align to set new styles to the table
  • tables dont have borders by default, U can set them by using an attribute called table border setting its value to 1
  • If a table cell has to span across multiple columns, U could use colspan attribute for that cell and allow it span across the columns
