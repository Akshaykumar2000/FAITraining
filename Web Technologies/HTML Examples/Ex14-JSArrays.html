Points to remember

  1. Arrays are of object type in JS
  2. It has a mix of collections and arrays concept in it. Arrays in JS allow to add, remove, update the data dynamically like Collections
  3. Newer version of JS also has MAP which works like Dictionary
  4. Some of the methods:
    • push: To add new elements to the bottom of the list
    • unshift: To add element to the begining of the list
    • shift: To remove the first element and return it.
    • splice: To remove the element
    • filter: to find elements on matching criteria
    • find: to find the first occurance of a single element on matching criteria
    • findIndex : To get the index of the matching Element
    • includes: to check if the element is in the list or not, it returns boolean
    • length: gets the no of elements within the array.
  5. There are 4 ways to iterate the elements of an array:
    1. for loop: Use the index to access the elements. for loop can also be used without the scope of a array. It allows to move forward as well as backward in the looping structure.
    2. for...of: Used to access the elements using iterator. Works similar to foreach statement. Introduced in ES7.
    3. for...in: Used to acceess the elements using index. Works similar to For loop, except for loop can work on non collections also and it can be reversed.
    4. Array.foreach: Used in earlier versions of JS where foreach is a function that takes a callback function for performing the iteration.
