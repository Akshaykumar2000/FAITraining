Points to remember

  1. HTML Document is programmatically represented as a Tree structure of DOM Elements. DOM stands for Document object Model
  2. Every element of the page is represented as a DOM Element and will be of a tree view hirarchy where U have children, siblings and parent elements.
  3. document is the root element/node of the Document. It represents the html document. Under document, U have elements in the form of a tree-view structure. It can be programmatically accessed by document object or window.document object. With document, we can access/refer any element of the document using JS programming, unless it is yet to be created.
  4. Element will have properties, methods and events which can be dynamically modified using JS.
