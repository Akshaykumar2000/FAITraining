Points to remember

  • Earlier versions of HTML did not support viewing of media files
  • We used to depend on 3rd party plugins like Adobe Flash and MS's Silverlight
  • With HTML 5, we can now access and view the media files like Audio as well as Video files
  • Mp3 and few more formats are supported for Audio and mp4 is supported for video files
